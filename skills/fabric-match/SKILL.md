# 面料匹配款式查询

## 功能说明

用户有现有面料库存，想知道能做什么款式。此skill从纸样数据库中匹配：
- 面料类型匹配（支持模糊匹配，如"棉麻"可匹配"棉麻、TR"）
- 米数充足性判断（按指定尺码判断用量是否足够）
- 输出所有符合条件的款式列表

## 数据源

纸样数据位于：`/home/DataTransfer/Pyrojewel/01_lab/excel_deal/extracted/styles.json`

每条记录包含：
- `style_id`: 款号
- `style_type`: 款式描述
- `fabric.suggestion`: 建议面料（可能是多种，如"棉麻、TR"）
- `fabric.usage`: 各尺码面料用量（XS-4XL）
- `fabric.width`: 面料幅宽
- `size_chart`: 尺码表

## 工作流程

### Step 1: 收集用户输入

用户会提供类似信息：
- 面料类型：如"棉麻"、"TR"、"府绸"
- 面料量：如"2米"、"2m*1.4m"（米数×幅宽）
- 目标尺码：如"M码"、"L码"

**如果信息不完整，主动询问**：
- 缺面料类型 → 问"你有什么面料？"
- 缺米数 → 问"你有多少米？"
- 缺尺码 → 问"你想做什么尺码？（XS/S/M/L/XL/2XL/3XL/4XL）"

### Step 2: 加载纸样数据

读取 `/home/DataTransfer/Pyrojewel/01_lab/excel_deal/extracted/styles.json`

### Step 3: 执行匹配

匹配逻辑：

1. **面料类型匹配**：
   - 用户输入的面料类型（如"棉麻"）
   - 与记录中的 `fabric.suggestion` 进行匹配
   - 支持模糊匹配：如果建议面料包含用户面料，则匹配成功
   - 例如：用户"棉麻" → 匹配"棉麻、TR"、"棉麻"、"棉麻、府绸"

2. **米数判断**：
   - 获取目标尺码的面料用量（如M码用量）
   - 判断用户米数 ≥ 用量，则可制作
   - 注意幅宽：如果用户提供了幅宽，需与 `fabric.width` 比较

3. **里布判断**（可选）：
   - 如果款式需要里布，提醒用户
   - 显示里布用量

### Step 4: 输出结果

输出格式：

```
## 匹配结果

**你的面料**：棉麻 2米（幅宽1.4米）
**目标尺码**：M码

找到 **X** 款可制作：

### 1. 小红叶829 - 船领连衣裙
- 面料用量：M码需 2米
- 幅宽要求：1.4米
- ✅ 你的面料刚好够用
- 里布需求：1.9米（需另备）

### 2. 小红叶828 - 翻领短袖上衣
- 靎料用量：M码需 1.2米
- ✅ 你的面料充足（剩余0.8米）
- 辅料：扣子5颗

...

### ❌ 不匹配的款式示例

- 小红叶830：建议面料为莱卡、针织，与你的棉麻不符
- 小红叶826：M码需2.5米，你的2米不够
```

## 边界情况处理

| 情况 | 处理方式 |
|------|----------|
| 面料类型不在库中 | 提示"暂无匹配该面料的款式"，列出库中常见面料供参考 |
| 所有款式米数都不够 | 提示"你的米数偏少"，列出最接近的款式（差多少米） |
| 用户没说尺码 | 默认按M码计算，或询问 |
| 幅宽不匹配 | 提示幅宽差异，说明可能影响裁剪 |

## 常见面料类型参考

库中常见面料：棉麻、TR、府绸、莱卡、针织、亚麻、雪纺、真丝棉、速干、鹿皮绒、人棉、洗水棉

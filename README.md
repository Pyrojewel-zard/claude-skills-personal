# Skill Manager

This repository contains a curated set of locally maintained skills for research, literature workflow, Zotero/CNKI/IEEE tasks, and the Obsidian LLM Wiki workflow.

## Current Scope

- CNKI search, parsing, download, and export
- IEEE Xplore search, metadata extraction, download, and research workflow
- Zotero item management and DOI import
- Daily notes, memo capture, and the three-entry LLM Wiki flow
- Selected research and experiment utilities

## LLM Wiki Flow

The accepted user-facing flow is:

`wiki-capture -> wiki-compile -> wiki-refine`

- `wiki-capture`: append low-density material into `raw/`
- `wiki-compile`: compile stable material into readable wiki pages
- `wiki-refine`: optionally promote high-value knowledge into typed nodes
- `wiki-ingest`: underlying compile engine
- `wiki-crystallize`: candidate review and promotion

# Skill Manager

This repository contains a curated set of locally maintained skills for research, literature workflow, and Zotero/CNKI/IEEE tasks.

## Current Scope

- CNKI search, parsing, download, and export
- IEEE Xplore search, metadata extraction, download, and research workflow
- Zotero item management and DOI import
- Daily notes, memo capture, and wiki ingest
- Selected research and experiment utilities

- `skills/cnki-*`
